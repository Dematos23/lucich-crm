# **App Name**: Lucich CRM

## Core Features:

- Authentication: Secure user authentication with email/password (Firebase Auth). Roles: admin, advisor, viewer.
- Client Management: Manage clients (companies), contacts, and related information.
- Opportunity Tracking: Track opportunities through different stages (lead, diagn√≥stico, etc.).
- Funnel Visualization: Visualize opportunities by stage in a funnel/pipeline view (Kanban).
- File Management: Upload and manage files (PDFs, images) related to clients and opportunities (Firebase Storage).
- Quote Comparison Tool: Compare quotes and generate a clear recommendation.
- Role-Based Access Control: Control access based on user roles (admin, advisor, viewer).

## Style Guidelines:

- Primary color: TL Gold (#F0B802) to reflect the brand's confidence, and for CTAs.
- Background color: Warm Gray (#F3F4F6) for a soft, neutral backdrop.
- Accent color: Deep Navy (#0B1F3B) for secondary CTAs, headers, and footers.
- Font: 'Source Sans 3' (sans-serif) to provide a readable, functional, and clear aesthetic across the CRM application.
- Base font size: 16-18px with line-height 1.5-1.7 for excellent legibility.
- Minimalist icons for navigation and actions, ensuring clarity and ease of use.
- Mobile-first, responsive design with sidebar navigation for desktop and topbar for mobile. Focus on reducing clicks and form lengths for efficiency.
- Subtle transitions and animations for UI states (loading, success, error).
